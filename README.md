#ScobyMovie

Continuación del trabajo del curso de Consumo de API REST con JavaScript
API : TMDB The Movie
Información de películas en tendencia y otras para descubrir. 
- Flitros por categorías y búsqueda. 
- Lazy Load para los posters de cada película.
- Scroll Infinito, utilizando la paginación de al API
- Skeleton Loader para el tiempo de carga.
